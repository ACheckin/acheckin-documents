:::note
Các loại app mà ACheckin cung cấp cho bạn
:::

## App Global
> Loại app chỉ dùng được trên `PERSONAL` space

:::danger Quan trọng
Để có thể sử dụng được app `GLOBAL` bạn cần phải được `ACheckin` xét duyệt
:::

Bạn cần thêm loại app phía dưới vào file [config.json](config) của mình
```json
{
    // ...
    "app_type": "global"
}
```
