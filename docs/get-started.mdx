## Cài đặt ACheckin cli

1. Để cài đặt ACheckin cli hãy chắc chắn bạn đã cài đặt [Node](https://nodejs.org/)

    > ACheckin khuyên bạn nên dùng phiên bản [Node](https://nodejs.org/) > 10. Bạn có thể cài đặt [Node](https://nodejs.org/) thông qua một package có tên là [nvm](https://github.com/nvm-sh/nvm) để có thể quản lý các phiên bản của [Node](https://nodejs.org/) một cách tốt nhất

    ```sh
        curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
    ```

2. Cài đặt ACheckin cli ở chế độ global
    ```bash npm2yarn
        npm install -g acheckin
    ```
